HOW TO START MINING

Step 1 - Create a wallet

Go to Gekkoin registration page.

*Gekkoin - a partner project that provided a service for storing, selling and transparently converting cryptocurrency into fiat assets

Step 2 - Download mining
software

Download XMRig (pass 12345) or compile from Source Code for mining Monero (calculete RandomX hashes).

We will add algorithms whenever possible.

Step 3 - Edit the bat file
and start mining

Edit your start bat file - write Gekkoin account and server address de.hashpower.online:7778.

Start mining.
